# Tkinter Password Manager

A simple password manager built with Python and Tkinter. This project is part of the "100 Days of Code" course.

## Features

- Generate strong random passwords
- Save credentials (Website, Email, Password) locally
- Graphical User Interface (GUI) with Tkinter
- Input validation and warning messages
- Passwords hidden in the entry field
- Easy-to-use buttons for generate and save

## How to Run

1. Clone the repo
2. Create and activate a Python virtual environment
3. Install required packages (if any)
4. Run the project:

```bash
python main.py
